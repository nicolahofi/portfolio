# Projekt-Dokumentation

## Überblick
Dieses Repository enthält meine Portfolio-Seite. Es ist ein reines Frontend-Projekt (HTML, CSS, JS) ohne Build-Stack.

## Projektbeschrieb
- Hero mit Carousel und CTA
- Audio- und Web-Projektübersicht mit Modal-Fenster
- About-Bereich inkl. Skills, CV und Kontakt-Buttons (E-Mail Copy, LinkedIn)
- Impressum

## Schwierigkeiten & Herausforderungen
- Responsives Header-Layout mit Burger-Menü und Desktop-Kompatibilität
- Hero Carousel (Desktop vs. Mobile Positionierung, Bildvarianten)
- Info-Panels für Projekte (Zugänglichkeit, Aria-Attribute)
- Kopierfunktion für E-Mail-Adresse

## Known Bugs / TODOs
- Hero Carousel: Muss evtl. für mobile Ansicht optimiert werden (Bildgrössen, Performance)
- Mobile Navigation: Bei sehr langen Link-Texten kann Navigation umbrechen
- Modal Accessibility: Fokus-Trap fehlt
